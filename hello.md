## git 명령어 요약

    - clone: 원격 저장소 복사
    - add: 스테이지 영역에 작업 파일 추가
    - commit: 세이브, 스테이지 영역의 파일들을 가지고 커밋(=세이브)을 만듦
    - push: 원격 저장소에 커밋을 업로드

## 파일 내용 되돌리기
    - 특정 파일 내용을 마지막 커밋으로 돌리고 싶다면 해당 파일 선택후 '코드 뭉치 버리기'선택


## 브랜치 변경
    - 브랜치: 기존 내용을 유지한 상태에서 새로운 내용 추가하고 싶을때 사용함
    - 체크아웃: 특정 브랜치로 돌아가고 싶을때 사용
